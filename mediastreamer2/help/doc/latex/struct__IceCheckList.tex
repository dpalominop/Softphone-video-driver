\subsection{Detailed Description}
Structure representing an I\-C\-E check list.

Each media stream must be assigned a check list. Check lists are added to an I\-C\-E session using the \doxyref{ice\-\_\-session\-\_\-add\-\_\-check\-\_\-list()}{p.}{ice_8h_a55f6051dac102bd53186ca91b80f6d10} function.
